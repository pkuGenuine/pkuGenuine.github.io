# 第一课

## 基本语法

### 类别 & 身份表述

基本结构： A ( 名 ) は B ( 名 ) です

肯定陈述：

1. 私は中国人です

否定陈述：

1. 林さんは学生 < では | じゃ > ありません

> じゃ 是 では 口语化的替代形式

提问：

1. 田中さんは会社員ですか

回答：

1. はい、そうです
2. いいえ、ちがいます



# 第二课

## 基本语法

### これ、それ、あれ、どれ

都是指示代词，これ 相当于“这”，それ、あれ 相当于“那”

こ 表示说话人附近的：

1. これは本です

そ 表示对方附近的：

1. それは何 ( なん ) ですか

あ 表示对话双方以外的：

1. あれは誰の傘ですか

どれ 用于提问，表示哪一个:

1. 長島さんの傘どれですか 

### この、その、あの、どの
相对于“这里的，那里的”

この、その、あの, どの 后面一般不可省略：

1. このカメラは田中さんのです
2. 長島さんの傘 < どれ | どの傘 > ですか     　　


# 第三课

## 基本语法

### ここ、そこ、あそこ、どこ
相当于“这里，那里，哪里”：

1. あそこは入り口です
2. かばんはどこですか

### 位置说明
基本结构： A ( 名 ) は B ( 场所 ) です

指示位置，但一般不用于人：

1. 食堂はデパートの七階です

> 小野さんは事務所です　也对，但一般不这么说

### も
也，和 は 不同时出现：

1. 李さんは教授です、張さんも教授です

### 多选项提问
基本结构： A は B ですか、 C ですか ...

答案多种可能，询问其中一种时，可重复使用谓语 ですか：

1. かばん売り場は一階ですか、二階ですか

### いくら
用以提问价格：

1. この服はいくらですか

### こちら、そちら、あちら、どちら

礼貌用语

可以和 ここ 等相替换：

1. 受付はどちらですか　—— あちらです
2. こちらは林さんです
3. どちらさまですか ( 您哪位 )

也可以加 の 替换 この 等:

1. こちらの方は森さんです

### お、ご
有表示礼貌的意思：

1. お国
2. お宅
3. お母さん
4. ご兄弟
5. ご両親

### ですか
ですか 语调升时表示提问，语调降时表示“啊”：

1. 大学ですか ( 升：是大学吗？降：是大学呀。 )
2. そうですか ( 升：是这样吗？降：是这样呀。 )

# 第四课

## 基本语法

### 存在句

基本结构：

1. A ( 场所 ) に B ( 物 ) が < あります | います >
2. A ( 物 ) は B ( 场所 ) に < あります | います >

あります 用于无意志的东西：
1. ここに本があります
2. 庭に何 ( なに ) がありますか
3. 机は部屋にあります

います 用于有意志的东西：
1. 公園に子供がいます
2. あそこに誰がいますか
3. 猫は部屋にいます
4. 犬はどこにいますか ( 与 犬はどこですか 等价 )

> 应该是陈述的时候尽量用存在句，提问的时候用简单的语法就好

### と

と 用在两个并列的名词间，表示“和”：

1. コーラとケーキは冷蔵庫にあります

### 方位词

基本结构：名 の 方位

1. 机の上に猫がいます
2. 会社の隣に花屋があります

> 日语的方位词相比于中文会更加明确，比如 上 ( うえ ) 就是指相对于重力的方向，不具有表面的意思

### ね
用在句子结尾，语调上扬时，一般是卖萌词，相当于“ xxx 对不对呢”。

そですね ( 语调向下 ) 可以表示“好呀”，不算上述卖萌情况。
### 全盘否定

基本结构：疑问句 も 动词 ( 否定 )

1. 教室に誰もいません
2. 冷蔵庫に何 ( なに ) もありません

> 注意疑问词后面的助词由 が 变成 も  

# 第五课

## 动词
动词都以 う 段结尾，分为三类：
1. 五段动词
2. 上一段动词，下一段动词
3. カ 变动词， サ 变动词

动词会因时态和否定而变形，不同类型的动词有不同的变形规则。目前可以认为有动词有五种形态：
1. 原型
2. 一般肯定 ( ます )
3. 一般否定 ( ません )
4. 过去肯定 ( ました )
5. 过去否定 ( ませんでした )


### 三类动词
三类动词范围较小：

1. カ 变动词只有一个词 来る
2. サ 变动词，也只对应一个词 する

する 表示“做”，可以加在名词后面，把名词“变成”动词：

1. 勉強する
2. 運動する

不是所有名词都可以加 する 变成动词，具体可以看词典。

变形方法：

1. 来 ( く ) る、来 ( き ) ます
2. する、します

### 一类动词
包括如下形式：
1. 结尾 う 段非 る
2. あ 段 + る | う 段 + る | お 段 + る
3. 特例，如 切る、帰る、走る

> 第二类中要除掉三类动词的情况

变形方法 ( ます 型 ) 为把结尾的 う 段音变成同行的 い 段音，再加 ます

1. 書く、書きます
2. 帰る、帰ります

### 二类动词
形式为 < い 段 + る | え 段 + る > ( 除去一类中特例的情况 )

变形方法为把 る 变成 ます

1. 食べる、食べます
2. 起きる、起きます

## 基本语法

### 表示动词发生时间

基本结构：时间 [ に ] 动词

这里的 に 不是可选，而是根据时间词可能需要加，也可能不能加：
1. 森さんは七時に起きます
2. 私は昨日勉強ませんでした

今、昨日、今日、明日、毎日、去年、来年 后面不加 に

月曜日 一类，有时候加，有时候不加

另一种结构，可以把时间词放在句首，用 、 隔开，就不虚要考虑是否加 に 了：

1. 昨日の八時半、私は起きました

### 表示时间的起始与结束

基本结构：时间 ( A ) から 时间 ( B ) まで

表示从 A 到 B 的这段时间。两个词也可分开使用：

1. 林さん9時から5時まで働きます
2. 王さんは土曜日まで勉強します

### 时间提问

基本结构：< いつ | 何時に | 何曜日に | 何日に > 动词 ですか

当然也可以使用 から 和 まで

1. 試験はいつ始まりますか
2. 仕事は何時に終わりますか
3. 展覧会はいつから始まりますか

### 时间词连接
一般用 の 就可以，但当前一个词带“每”的时候，则不能加 の

1. 一昨日の9時、私は起きました
2. 毎日7時、楓原さんは起きます

> 貌似 午前 和 午後 也不加

### 时间大约
用 ごろ 表示“左右”，后面不加 に

1. 一時三十分後ろ ( 不能太准确，一般就精准到半点 )
2. 二時半ごろ

# 第六课

## 基本语法

### 用 へ 表示行动目的地
基本结构：地点 へ 动词

1. 吉田さんは中国へ行きます ( 去中国 )
2. 林さんは日本へ帰りました ( 回日本 )
3. 森さんはどこへ行きますか

### 用 から 表示行动起点

1. 李さんは先月北京から来ました

### 用 と 表示共同做某事的人

1. 小野さんは友達と帰りました
2. 田中さんは誰と日本へ来ますか ( 和谁一起来日本呢 )

### 用 で 表示行动的交通工具 ( 方法 )
1. 北海道まで飛行機で行きます
2. 京都へ何 ( なに ) できましたか

如未使用交通工具，用 歩いて 表示步行

1. 李さんは歩いてアパートへ帰りました

### 用 たしか 表示不确定

1. あの人は誰ですか —— たしか、王さんです

## 单词解释

### 家
いえ 强调房子，而 うち 强调家庭。表达我家的时候，使用 私のいえ 或者 うち

### まっすぐ
直接去，没有绕路去做别的事

1. 昨夜はまっすぐ帰りましたか

# 第十課

## 第二类形容词
类似名词。

今日は火曜日です、昨日は月曜日でした、今日は水曜日ではありません、昨日は木曜日ではありませんでした

京都の紅葉は有名です、この町はにぎやかでした、この通りは静かではありません、昨日は暇じゃありませんでした

今日は水曜日ではない、今日は水曜日じゃない、昨日は木曜日ではなかった

( 感觉是 ない 的位置和一类不一样，一类在 です 前面，二类在 では 后面 )
